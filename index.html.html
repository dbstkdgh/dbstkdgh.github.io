노포식당 검색기

수원시 노포 식당 정보를 검색하는 웹 애플리케이션

Live Demo (준비중) GitHub

프로젝트 소개

목적: 수원시의 전통 노포 식당 정보를 JSON 데이터셋을 활용해 검색할 수 있는 웹 기반 애플리케이션입니다.

주요 기능: 식당명, 주소, 메뉴, 가격으로 검색 가능하며, Streamlit을 활용한 사용자 친화적 인터페이스를 제공합니다. 메뉴 검색 시 가격 필터링 기능과 URL 디코딩된 주소 출력 기능을 지원합니다.

#기술 스택
Python Streamlit JSON RegEx

나의 역할

  • 개발자: 데이터 수집, 전처리, 검색 기능 구현, 사용자 인터페이스 설계 등 애플리케이션 전반의 개발을 담당했습니다.
  • UI/UX 설계자: Streamlit을 활용해 직관적인 인터페이스를 설계하고, 사이드바를 통한 사용 안내 및 검색 결과 출력 레이아웃을 구성했습니다.

배운 점 (Lessons Learned)

  • 데이터 처리: JSON 데이터 파싱 및 처리 능력이 향상되었고, 가격 데이터의 예외 상황(누락/비정상 데이터)을 정규 표현식으로 처리하는 기술을 습득했습니다.
  • Streamlit 웹 개발: Streamlit을 활용한 동적 웹 애플리케이션 구축 경험을 쌓았고, 컬럼 및 사이드바 레이아웃, 사용자 입력 처리 기술을 학습했습니다.

성과

  • 완성도 높은 검색 도구: 식당 명, 주소, 메뉴, 가격 별 검색을 지원하는 완성도 높은 검색 엔진을 구현했습니다.
  • 사용자 경험 개선: URL 디코딩을 통한 주소 가독성을 향상시키고, 가격 입력 오류 처리 로직으로 사용자 편의성을 강화했습니다.
