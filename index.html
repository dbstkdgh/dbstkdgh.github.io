#기술 스택

Python Pandas Numpy Scikit-learn FastText eKoNLPy Kiwipiepy Matplotlib Seaborn NLP Text Mining Naive Bayes N-gram TF-IDF CountVectorizer

프로젝트 소개

경제 뉴스 기사, 한국은행 금융통화위원회(MPB) 의사록, 금융 보고서 등 비정형 텍스트 데이터 텍스트 마이닝 기법으로 분석, 금리 변화 방향 예측 모델 개발.

목적

중앙은행 커뮤니케이션의 매파적/비둘기파적 '톤' 정량화, 예측 모델에 활용하여 금리 인상 및 하락 예측 정확도 향상. 텍스트 기반 지표가 기존 거시경제 변수만으로는 포착하기 어려웠던 정책 결정의 추가적인 설명력 제공 입증.

프로젝트 수행 과정

  • 대규모 경제 텍스트 데이터 수집: 연합인포맥스, 이데일리, 한국경제 뉴스 및 한국은행 금통위 의사록 등 총 20만 건 이상 금리 관련 텍스트 데이터 수집.
  • 고도화된 한국어 텍스트 전처리: 품사 필터링, 불용어 제거, 띄어쓰기 교정 등 한국어 특성 최적화 전처리 파이프라인 구축.
  • 매파/비둘기파 감성 사전 구축 및 분류: eKoNLPy, Kiwipiepy, FastText 활용 텍스트 토큰화 및 단어 임베딩. 매파적(Hawkish) 성격과 비둘기파적(Dovish) 성격을 나누는 감성 사전 구축. 코사인 유사도 기법으로 개별 단어 분류, 의사록(정답셋) 감성 톤 예측.
  • 금리 예측 모델링: N-gram 특징 (1~5그램)과 Naive Bayes Classifier(NBC) 중심으로 CountVectorizer, TF-IDF 기법 활용 금리 변화 예측 모델 개발.
  • 탐색적 데이터 분석(EDA) 및 시각화: 총 수집 기사 수, 데이터 처리 파이프라인, FastText 단어 임베딩 산점도, 각 모델 학습 결과 및 성능 비교 등 시각적 분석 자료 제공.

역할

  • 데이터 수집 및 전처리 기여: 약 3만 건에 달하는 연합인포맥스 뉴스 기사 크롤링 및 전처리 과정 참여. 고품질 데이터 정제 작업(품사 필터링, 불용어 제거, 띄어쓰기 교정) 수행.
  • 텍스트 임베딩 및 모델 학습 참여: FastText 모델 학습 참여, 서브워드 학습, OOV(Out-Of-Vocabulary) 처리, Rare Word(저빈도 단어) 대응 등 임베딩 성능 최적화 기여.

성과

  • 고성능 금리 예측 지표 개발: 매파/비둘기파 감성 사전 구축 및 코사인 유사도 기반 분류 통해 한국은행 통화정책 '톤' 성공적 정량화. 텍스트 기반 지표가 기존 거시경제 변수 대비 한국은행의 통화정책 결정에 대한 높은 설명력 입증.
  • 강화된 예측 모델 성능: 텍스트 마이닝을 통해 얻은 정량적 정보가 금리 인상/하락 예측 모델의 설명력과 예측 정확도 유의미하게 개선. (CountVectorizer Accuracy 0.60, TF-IDF Accuracy 0.58).
  • 텍스트 데이터 활용 능력 증명: 대규모 비정형 텍스트 데이터 수집, 전처리, 임베딩, 모델링 전 과정 숙련된 기술 활용 및 복잡한 문제 해결 역량 증명.

레슨 런 (Lessons Learned)

  • 한국어 자연어 처리의 중요성: 한국어 텍스트의 복잡한 특성 고려 eKoNLPy 등 경제·금융 특화 NLP 도구 활용이 분석 정확도 극대화에 필수적임을 체감.
  • 분야별 감성 사전 구축의 필요성: 범용 사전으로 포착 어려운 경제·금융 분야 미묘한 의미 차이 정확히 반영 위해 특정 분야 특화 감성 사전 구축 및 유사도 기반 분류의 중요성 깊이 학습.
  • 모델 선택 및 최적화 전략: Word2Vec, FastText 등 다양한 단어 임베딩 기법과 N-gram, NBC 등 모델링 기법의 특성 이해 및 실제 데이터에 최적화된 조합을 찾아 예측 성능 지속적 개선 경험.

목적

이상동기 범죄 발생 가능성 사전에 파악, 지역 주민과 당국에 실시간 경보 및 지역별 위험 정보 제공으로 범죄 예방과 공공 안전 강화. 기상, 경제, 검색량 데이터 결합으로 범죄 발생 환경적·사회적 요인 정량화 및 예측 정확도 향상.

프로젝트 수행 과정

  • 다양한 데이터 수집 및 통합: 2015~2025년 범죄 데이터(시군구 단위), 기상청 데이터(습도, 기온 등), 네이버 데이터랩 검색량, 경제 지표(실업률 등) 수집. 약 10년간의 데이터 통합 분석 기반 구축.
  • 데이터 전처리 및 매핑: 지역 데이터 광역시도 단위 재분류, 기상 및 범죄 데이터 250×250m 격자 단위 정규화. 결측값 처리 및 한국어 특성 맞춤 검색량 데이터 전처리.
  • 예측 모델 개발: RandomForest, XGBoost with SMOTE, LSTM, ARIMA 활용 범죄 발생 가능성 예측. SMOTE로 데이터 불균형 문제 해결, LSTM으로 검색량 시계열 데이터 분석 시간적 패턴 도출.
  • Streamlit 대시보드 구현: Folium과 GeoJSON 활용 지역별 위험도 인터랙티브 지도 시각화. 4단계 위험 경보 체계(Safe, Warning, Danger, Very Dangerous) 임계값(0.6, 0.7, 0.85) 설정으로 사용자 친화적 대시보드 제공.
  • 탐색적 데이터 분석(EDA) 및 시각화: 기상 데이터와 범죄 발생 간 상관관계, 검색량과 범죄 건수의 시차 분석(±3일, ±7일) 통해 패턴 분석. Plotly 활용 시계열 그래프 및 지역별 히트맵 생성.

역할 (팀 리더)

  • 데이터 전처리 및 통합: 2015~2025년 범죄, 기상, 검색량 데이터 수집, 광역시도 및 격자 단위 매핑. 결측값 처리와 데이터 정규화 주도 모델 학습 기반 마련.
  • 모델링 및 최적화 전문가: XGBoost와 SMOTE 활용 이진 분류 모델 개발, LSTM으로 시계열 데이터 분석. 하이퍼파라미터 튜닝과 임계값 최적화 통해 예측 성능 개선.
  • 대시보드 개발자: Streamlit과 Folium 사용 지역별 위험 경보 시각화. GeoJSON 기반 자치구 경계 지도 구현 및 4단계 경보 체계 설계.

성과

  • 정확한 범죄 예측 모델 개발: 기상, 경제, 검색량 데이터 통합 XGBoost와 LSTM 모델로 이상동기 범죄 발생 가능성 예측 성공. SMOTE 적용으로 소수 클래스 예측 성능 개선.
  • 사용자 친화적 대시보드 구현: Streamlit 기반 대시보드를 통해 지역별 위험도 직관적 시각화. 4단계 경보 체계로 실시간 정보 제공, 공공 안전 강화 기여.
  • 데이터 기반 인사이트 도출: 기상 데이터(습도, 기온)와 검색량 데이터의 시차 분석 통해 범죄 발생의 환경적·사회적 요인 규명. 상관계수(0.54~0.58) 기반 패턴 분석으로 예측 설명력 향상.

레슨 런 (Lessons Learned)

  • 다양한 데이터 통합의 중요성: 범죄, 기상, 경제, 검색량 등 이종 데이터 통합 및 정규화 과정에서 데이터 단위와 시간적 일관성 유지가 예측 성능에 큰 영향 미침 학습.
  • 모델 최적화의 필요성: SMOTE, 하이퍼파라미터 튜닝, 임계값 조정을 통해 데이터 불균형과 과적합 문제 해결하며 모델 성능 최적화 경험.
  • 시각화의 효과: Folium과 GeoJSON 활용 인터랙티브 지도와 Plotly 기반 시각화로 복잡한 데이터를 사용자 친화적으로 전달하는 중요성 체감.
